    2.1.9 Define the terms: bit, byte, binary, denary/decimal, hexadecimal

    Bits and Bytes

    • Bit
      • Smallest unit of data in computer
      • Has a single binary number, either 0 or 1

    • Byte
      • A group of 8 binary bits operated on as a unit

    • Nibble
      • Half byte, a four-bit aggregation

    Number Systems

    • Binary
      • Has base-2 numbering, because there are only 2 digits (0 & 1)
      • Computers use binary
    • Decimal (Denary)
      • Base-10 numbering system
      • Can be anything based on number 10
    • Hexadecimal
      • Base-16 numbering system
      • Has 0 to 9 then A-F
      • Usually used in colour representation
