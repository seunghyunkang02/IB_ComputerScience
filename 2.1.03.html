    2.1.3 Explain the use of cache memory

    Cache Memory

    Cache is a hardware or software component that stores data so that data can be served faster for the future request.
    The data stored in a cache might be the result of an earlier computation or a copy of data stored elsewhere.

    • Level 1 and Level 2 Cache
      • Level 1 Cache - within the microprocessor
      • Level 2 Cache - outside of the microprocessor, on the motherboard
