    Domain Name System (DNS)
    C.1.6 Describe how a DNS server works

    • User requests a webpage using URL
    • URL is passed on the DNS server of ISP
    • DNS server returns either IP address from its cache or interrogating other DNS servers or an error
    • After DNS returns the IP addresses, IP addresses requests the required page
    • Requested page will be pass onto ISP and then to the user

    During the process of the user requesting a web page with URL, IP address is obtained from:
    Browser's Cache, Cache of the ISP's DNS or other DNS servers

    ERROR is returned if DNS record which links the required URL with an IP address does not exist
